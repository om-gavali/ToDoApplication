# ToDo Application

A simple ToDo Application built using Spring Boot that allows users to manage daily tasks efficiently.

## Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- RESTful API design

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- REST APIs
- MySQL / H2 Database
- Maven

## Project Structure
- Controller layer for handling requests
- Service layer for business logic
- Repository layer for database operations
